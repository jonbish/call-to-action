== Description ==

Displays the most relevant Call to Action in your sidebar based on the content of the page.

Calls to action are displayed randomly based on the category they are assigned to. If no category is assigned to a Call to Action then that action will display on pages with no categories.

The plugin also displays impressions and clicks.

== Installation ==

1. Upload the 'call-to-action' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add new Actions and assign them categories to display within
1. Drag the Call to Action widget to your sidebar or use the shortcode [ctaw] within your posts/pages

== Screenshots ==

1. Actions list with impressions and click counts
2. Edit Action page allows you to upload images and create links easily
3. Can be displayed as a widget in your sidebar or in your content as a shortcode
